《洋蔥燒豬肉》

《食材》
  • 洋蔥 2顆
  • 豬胛心肉 200克
  • 蔥 2枝
  • 醬油 1大匙
  • 素蠔油 1大匙
  • 味醂 1大匙
  • 醬油 1大匙
  • 米酒 1大匙
  • 白胡椒粉 1/2大匙
《步驟》
  1. 豬肉切塊加入醃料拌勻 (豬肉最好選有帶肥肉的比較不會老，另外用肉片也可以)
  2. 洋蔥切絲下鍋炒香，炒軟後放入豬肉和蔥段拌炒。
  3. 炒香後加入所有調味料和水(淹過食材)用中小火慢慢燉煮
  4. 煮到洋蔥完全軟爛後就可以關火了 (要留一點湯汁比較好吃喔)
  5. 完成。
