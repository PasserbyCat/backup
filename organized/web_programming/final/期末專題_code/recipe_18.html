《三杯杏鮑菇梅花豬》

《食材》
  • 薑一塊(約15g)切薄片
  • 蒜兩瓣 切末
  • 杏鮑菇1-2根 切段再切花刀
  • 辣椒半根 切片
  • 九層塔去除硬梗
  • 醬油 兩大匙
  • 米酒 適量
  • 糖 一大匙
  • 麻油 適量
《步驟》
  1. 鍋燒熱，放進杏鮑菇煎至金黃。
  2. 梅花豬肉片放進鍋中快速煎一下（會出油）。
  3. 豬肉的油不要擦掉，補入一點點橄欖油和黑麻油，放入薑片爆香至呈現捲曲。
  4. 放入蒜末，接著放入剛剛的豬肉片和杏鮑菇拌炒，加入一匙糖炒出糖色。
  5. 加入醬油、米酒快速拌炒至收汁。
  6. 起鍋前加入九層塔、辣椒即可。
  7. 完成。
《寄送食譜》
