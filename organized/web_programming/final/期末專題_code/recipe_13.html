《香烤豬後腿肉佐生洋蔥》

《食材》
  • 豬後腿肉一盒
  • 洋蔥 1/4顆
  • 醬油 2大匙
  • 米酒 1大匙
  • 糖 1小匙
《步驟》
  1. 將豬肉洗淨，加入醬油、米酒、糖，醃漬20分鐘。
  2. 烤盤鋪鋁箔紙，將醃漬好的豬肉一一擺放上去，約烤5分鐘後在豬肉上方再鋪一層鋁箔紙 (目的是隔開豬肉與上方燈管，避免表面烤焦，內部不熟的狀況) ，繼續烤10分鐘。
  3. 用筷子試試若是可以順利戳穿即可取出。
  4. 生洋蔥切絲，豬肉切片擺盤。
  5. 完成。
《寄送食譜》
