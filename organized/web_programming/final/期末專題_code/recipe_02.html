《冰糖花生豬腳》

《食材》
  • 豬腳 1200 公克
  • 花生 200 公克
  • 薑片 5 片
  • 蒜仁 5 瓣
  • 八角 1 粒
  • 米酒 1 大匙
  • 醬油 90 cc
  • 水 300 cc
  • 冰糖 2大匙
《步驟》
  1. 花生洗淨後先用滾水泡約30分鐘，然後瀝乾備用。
  2. 準備一鍋滾水，加入米酒，將豬腳汆燙去血水。
  3. 將豬腳、花生及其它所有材料放入電鍋，稍微翻拌一下讓食材都沾裹到醬汁。
  4. 入電鍋外鍋2杯水跳起2次，再悶30分鐘。
  5. 完成。
《寄送食譜》
