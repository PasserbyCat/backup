《脆皮德國豬腳》

《食材》
  • 冷凍德國豬腳 一支約700克
  • 洋蔥一顆
  • 義式香料
  • 黑胡椒
  • 鹽
  • 沙拉油 一小匙
《步驟》
  1. 準備一鍋滾水，將豬腳汆燙去血水。
  2. 蔥切成八等分(底層不要切斷)，調味油混合後抹上洋蔥(內外皆塗上)，用錫箔紙包起。
  3. 烤箱預熱250度，烤約30~40分分鐘到豬腳表皮焦香酥脆(表皮也可抹一點油);洋蔥可視個人喜好在烤到20分鐘時取出，如果沒有提前取出口感會變得很軟。
  5. 腳用刀去骨後切小塊，烤到香脆外皮的德式豬腳搭配黃芥末。
  6. 完成。
