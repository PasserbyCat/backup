《椒鹽松阪豬》

《食材》
  • 松阪豬 1 片
  • 洋蔥 1/4 顆
  • 蔥 1 根
  • 蒜末 2 顆
  • 檸檬 1 顆
  • 黑胡椒 適量
  • 鹽 適量
  • 白胡椒 適量
  • 香油 適量
《步驟》
  1. 洋蔥和蔥切成小段混和，加入些許黑胡椒、鹽、香油和檸檬汁。
  2. 攪拌均勻後試味道，完成後放入冰箱。
  3. 松阪豬表面抹點鹽和白胡椒粉，用刀子劃幾刀(煎的時候比較不捲曲)。
  4. 鍋內加入一點點香油和橄欖油，放入蒜末炒香，放入松阪豬，中火蓋鍋，數分鐘後翻面，煎到全熟。
  5. 完成。
