《泡菜炒豬肉》

《食材》
  • 米酒 1大匙
  • 醬油 1大匙
  • 太白粉 1大匙
  • 胛心肉 100g逆紋切薄片
  • 蒜末 4瓣
  • 高麗菜 1/8顆
  • 蔥 一支切段
  • 泡菜 半罐（汁也要留）
  • 醬油或醬油膏 1匙
  • 糖 1匙
《步驟》
  1. 豬肉逆紋切薄片，將肉跟醃料先混合拌勻，醃一下。高麗菜切絲、蔥切段、蒜頭切末。
  2. 炒鍋中倒入一點油。將肉先炒熟或過油皆可，接著肉撈起放一旁備用。
  3. 原鍋中爆香蒜末，再倒入泡菜（含汁）及高麗菜，加入約1/4杯水，煮到高 麗菜軟化。
  4. 放入肉片及調味料拌勻，起鍋前放入蔥段。
  5. 完成。
《寄送食譜》
