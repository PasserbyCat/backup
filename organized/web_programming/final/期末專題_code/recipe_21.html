《蜂蜜檸檬豬腱肉》

《食材》
  • 豬腱肉（後腿腱） 1塊(約300g)
  • 蜂蜜/檸檬汁/醬油 各20g
  • 蒜 1 瓣
《步驟》
  1. 將所有調味料倒入乾淨塑膠袋中揉勻，再加入蒜頭切片。
  2. 放入豬腱肉揉勻，封上袋口冷藏醃漬至少1天。
  3. 不沾鍋免倒油，直接放入退冰好的豬腱肉，蓋上鍋蓋以小火悶煎至肉泛白。
  4. 將豬肉翻面，倒入醃醬，蓋上鍋蓋續悶煎約8分鐘。
  5. 再次將豬肉翻面，續悶煎約2~3分鐘至肉熟即可。
  6. 完成。
《寄送食譜》
