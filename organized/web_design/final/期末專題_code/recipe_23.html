《照燒腱子肉》

《食材》
  • 腱子肉 1斤半
  • 蒜頭 適量
  • 辣椒 一根
  • 白芝麻 適量
  • 醬油 1/3 碗
  • 醬油膏 1/3 碗
  • 味琳 半碗
  • 水 一碗半
  • 酒 一碗
  • 冰糖 1 匙
《步驟》
  1. 材料洗淨。
  2. 起油鍋，將蒜瓣，辣椒炒香。
  3. 將豬腱肉放入炒。
  4. 依序加入醬油，醬油膏，味琳，冰糖，酒，水，能蓋過肉8分滿 蓋上鍋蓋煮至肉熟。
  5. 肉熟後撒上白芝麻即可盛盤。
  6. 完成。
