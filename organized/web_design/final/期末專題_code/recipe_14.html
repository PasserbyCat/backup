《空心菜沙茶後腿肉》

《食材》
  • 豬後腿切絲 150 g
  • 空心菜切段（梗葉分開）1 把
  • 蒜瓣切末 3 瓣
  • 醬油 少許
  • 胡椒鹽 少許
  • 米酒 少許
  • 太白粉 少許
  • 水（嗆鍋用）酌量
  • 沙茶醬 酌量
  • 烏醋 酌量
  • 鹽 少許
  • 鮮味炒手 少許
《步驟》
  1. 蒜瓣.空心菜洗淨分切備用。豬肉切絲抓醃料備用。
  2. 啟熱油鍋蒜末爆香後肉絲入鍋拌炒呈熟成狀先剷起備用。
  3. 接序將空心菜梗先煸炒後，肉絲回鍋沙茶醬.烏醋.水一起拌炒。
  4. 加入空心菜葉再調味拌炒一會兒即可盛食上菜。
  5. 完成。
《寄送食譜》
