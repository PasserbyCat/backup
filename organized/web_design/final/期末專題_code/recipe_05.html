《板栗紅燒肉》

《食材》
  • 五花肉500克
  • 板栗20克
  • 薑適量
  • 老抽2湯匙
  • 料酒1湯匙
  • 食鹽少許
  • 冰糖10克
  • 小蔥1根
《步驟》
  1. 五花肉洗凈拭幹表面水份，切成小塊，姜切片蔥切段。
  2. 鍋燒熱，不放油，直接倒入五花肉，用中小火幹煸至肉表面上色，微微焦黃。
  3. 放入薑片蔥段炒出香味。
  4. 加入一湯匙料酒，兩湯匙老抽。
  5. 放入一小把冰糖（約10g），不斷翻炒至五花肉上色。
  6. 倒入足量水（用普通炒菜鍋煮的話，水量至少要和肉齊平或者沒過肉）
  7. 倒入栗子，中大火煮開。
  8. 轉小火加蓋燜煮40分鐘左右，至五花肉熟透，開蓋按個人口味加鹽調味，轉中火把鍋中湯汁收濃即可。
  9. 完成。
