《豬里肌肉片豆腐捲》

《食材》
  • 豬里肌肉 600克
  • 小黃瓜 1條
  • 紅蘿蔔 1條
  • 油豆腐 600克
  • 蒜頭 4顆
  • 薑片 3片
  • 豚骨高湯 1000克
  • 玉米粉 1大匙
  • 橄欖油 適量
  • 醬油 4大匙
  • 糖 1大匙
《步驟》
  1. 油豆腐切成長條狀，小黃瓜、紅蘿蔔切成長條薄片備用。
  2. 用豬里肌肉片將作法1捲好，並在肉捲接合處糊上玉米粉。
  3. 準備平底鍋，加入少許橄欖油燒熱後，將作法2煎至雙面均勻上色，起鍋備用。
  4. 在鍋中放入蒜和薑並爆香，加入豚骨高湯及醬油、糖，醬汁燒開後，將作法3下鍋紅燒，煮至入味即可上桌。
  5. 完成。
《寄送食譜》
